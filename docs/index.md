## Quick Example

```html
<div
  layout="flex col center"
  space="p:big"
  visual="bg:primary text:white rounded:big"
>
  Hello SenangStart!
</div>
```

This creates a flexbox column, centered, with big padding, primary background, white text, and big rounded corners. **No CSS classes to memorize.**
