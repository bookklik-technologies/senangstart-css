# Font Weight

Set font weight

## Syntax
```
visual="font:[weight]"
```

## Values

| Value | CSS Output | Description |
|-------|------------|-------------|
| `normal` | `font-weight: var(--fw-normal)` | Normal weight |
| `medium` | `font-weight: var(--fw-medium)` | Medium weight |
| `bold` | `font-weight: var(--fw-bold)` | Bold weight |

## Examples

```html
<div visual="font:bold">Bold text</div>
```
