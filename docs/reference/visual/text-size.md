# Text Size

Set font size

## Syntax
```
visual="text-size:[value]"
```

## Examples

```html
<div visual="text-size:big">Large text</div>
<div visual="text-size:[24px]">24px text</div>
```

## Arbitrary Values

Supports custom values using bracket syntax:

```html
<div visual="text:[custom-value]">Custom</div>
```

## Responsive

```html
<!-- Responsive example -->

```
