# Hyphens

Control hyphenation

## Syntax
```
visual="hyphens:[value]"
```

## Values

| Value | CSS Output | Description |
|-------|------------|-------------|
| `none` | `hyphens: none` | No hyphens |
| `manual` | `hyphens: manual` | Manual hyphens |
| `auto` | `hyphens: auto` | Auto hyphens |

## Examples

```html
<p visual="hyphens:auto">Automatic hyphenation</p>
```
