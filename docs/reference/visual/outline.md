# Outline

Set outline color

## Syntax
```
visual="outline:[color]"
```

## Examples

```html
<button visual="focus:outline:primary">Focus outline</button>
```

## Arbitrary Values

Supports custom values using bracket syntax:

```html
<div visual="outline:[custom-value]">Custom</div>
```

## Responsive

```html
<!-- Responsive example -->
<div visual="mob:... tab:... lap:...">
  Responsive content
</div>
```
