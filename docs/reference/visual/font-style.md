# Font Style

Set font style

## Syntax
```
visual="[style-value]"
```

## Values

| Value | CSS Output | Description |
|-------|------------|-------------|
| `italic` | `font-style: italic` | Italic text |
| `not-italic` | `font-style: normal` | Normal style |

## Examples

```html
<em visual="not-italic">Normal style emphasis</em>
```

## Responsive

```html
<!-- Responsive example -->
<div visual="mob:... tab:... lap:...">
  Responsive content
</div>
```
