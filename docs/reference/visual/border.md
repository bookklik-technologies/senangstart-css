# Border

Set border color

## Syntax
```
visual="border:[color]"
```

## Examples

```html
<div visual="border:primary border-w:small">Bordered element</div>
```

## Arbitrary Values

Supports custom values using bracket syntax:

```html
<div visual="border:[custom-value]">Custom</div>
```

## Responsive

```html
<!-- Responsive example -->
<div visual="mob:... tab:... lap:...">
  Responsive content
</div>
```
