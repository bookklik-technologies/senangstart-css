# Columns

Create multi-column layouts

## Syntax
```
layout="cols:[value]"
```

## Values

| Value | CSS Output | Description |
|-------|------------|-------------|
| `1-12` | `columns: {n}` | N columns |
| `auto` | `columns: auto` | Auto columns |

## Examples

```html
<div layout="cols:3">Three columns</div>
```
