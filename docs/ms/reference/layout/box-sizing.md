# Box Sizing

Mengawal bagaimana dimensi elemen dikira.

## Nilai

| Nilai | Output CSS |
|-------|------------|
| `box-border` | `box-sizing: border-box` |
| `box-content` | `box-sizing: content-box` |

## Contoh

```html
<div layout="box-border" space="p:medium w:[200px]">
  Padding termasuk dalam lebar 200px
</div>
```
