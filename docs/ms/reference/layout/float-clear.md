# Float & Clear

Mengawal tingkah laku float dan clear.

## Nilai

| Nilai | Output CSS |
|-------|------------|
| `float-left` | `float: left` |
| `float-right` | `float: right` |
| `float-none` | `float: none` |
| `clear-left` | `clear: left` |
| `clear-right` | `clear: right` |
| `clear-both` | `clear: both` |

## Contoh

```html
<img layout="float-left" space="m-r:medium" src="imej.jpg">
<p>Teks mengelilingi imej yang di-float...</p>
<div layout="clear-both"></div>
```
