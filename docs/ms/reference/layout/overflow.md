# Overflow

Mengawal tingkah laku overflow dengan atribut `layout`.

## Nilai

| Nilai | Output CSS |
|-------|------------|
| `overflow:hidden` | `overflow: hidden` |
| `overflow:auto` | `overflow: auto` |
| `overflow:scroll` | `overflow: scroll` |

## Contoh

```html
<div layout="overflow:hidden">Kandungan dipotong</div>
<div layout="overflow:auto">Boleh skrol bila perlu</div>
```
