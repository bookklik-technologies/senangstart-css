# Container

Cipta bekas berpusat dengan lebar maksimum

## Sintaks
```
layout="container"
```

## Nilai

| Nilai | CSS Output | Huraian |
|-------|------------|-------------|
| `container` | `width: 100%; margin-left: auto; margin-right: auto` | Bekas berpusat |

## Contoh

```html
<div layout="container">Centered content</div>
```

## Responsif

```html
<!-- Contoh responsif -->
<div layout="mob:... tab:... lap:...">
  Kandungan responsif
</div>
```
