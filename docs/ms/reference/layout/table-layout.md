# Table Layout

Kawal algoritma susun atur jadual

## Sintaks
```
layout="table:[value]"
```

## Nilai

| Nilai | CSS Output | Huraian |
|-------|------------|-------------|
| `auto` | `table-layout: auto` | Susun atur automatik |
| `fixed` | `table-layout: fixed` | Susun atur tetap |

## Contoh

```html
<table layout="table:fixed">Fixed width columns</table>
```
