# Columns

Cipta susun atur berbilang lajur

## Sintaks
```
layout="cols:[value]"
```

## Nilai

| Nilai | CSS Output | Huraian |
|-------|------------|-------------|
| `1-12` | `columns: {n}` | N lajur |
| `auto` | `columns: auto` | Lajur automatik |

## Contoh

```html
<div layout="cols:3">Three columns</div>
```
