# Object Position

Mengawal kedudukan imej/video dalam kontena.

## Nilai

| Nilai | Output CSS |
|-------|------------|
| `object-pos:center` | `object-position: center` |
| `object-pos:top` | `object-position: top` |
| `object-pos:bottom` | `object-position: bottom` |

## Contoh

```html
<img layout="object:cover object-pos:top" src="potret.jpg">
```
