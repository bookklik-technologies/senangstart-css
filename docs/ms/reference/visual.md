# Rujukan Visual

Atribut `visual` mengawal warna, tekstur, tipografi, dan penampilan.

## Warna

- [Latar Belakang](./visual/background) - `bg:white`, `bg:primary`, `bg:[#hex]`
- [Warna Teks](./visual/text-color) - `text:dark`, `text:grey`, `text:primary`

## Tipografi

- [Saiz Teks](./visual/text-size) - `text-size:small`, `text-size:big`, `text-size:giant`
- [Keluarga Fon](./visual/font-family) - `font:sans`, `font:serif`, `font:mono`
- [Berat Fon](./visual/font-weight) - `font:normal`, `font:medium`, `font:bold`
- [Gaya Fon](./visual/font-style) - `italic`, `not-italic`
- [Smoothing Fon](./visual/font-smoothing) - `antialiased`, `subpixel-antialiased`
- [Jarak Huruf](./visual/letter-spacing) - `tracking:tight`, `tracking:wide`
- [Ketinggian Baris](./visual/line-height) - `leading:tight`, `leading:relaxed`
- [Line Clamp](./visual/line-clamp) - `line-clamp:1`, `line-clamp:2`, `line-clamp:3`

## Pemformatan Teks

- [Penjajaran Teks](./visual/text-alignment) - `text:left`, `text:center`, `text:right`
- [Transformasi Teks](./visual/text-transform) - `uppercase`, `lowercase`, `capitalize`
- [Hiasan Teks](./visual/text-decoration) - `text-decoration-line:underline`, `text-decoration-style:wavy`
- [Limpahan Teks](./visual/text-overflow) - `truncate`, `text-ellipsis`
- [Pembalutan Teks](./visual/text-wrap) - `text-wrap:wrap`, `text-wrap:nowrap`, `text-wrap:balance`
- [Whitespace](./visual/whitespace) - `white-space:pre`, `white-space:nowrap`
- [Word Break](./visual/word-break) - `word-break:normal`, `overflow-wrap:break-word`, `word-break:break-all`
- [Hyphens](./visual/hyphens) - `hyphens:auto`, `hyphens:none`
- [Inden Teks](./visual/text-indent) - `indent:small`, `indent:medium`
- [Penjajaran Menegak](./visual/vertical-align) - `vertical-align:middle`, `vertical-align:top`

## Senarai

- [Gaya Senarai](./visual/list-style) - `list-none`, `list-disc`, `list-decimal`
- [Varian Nombor Fon](./visual/font-variant-numeric) - `tabular-nums`, `ordinal`

## Sempadan & Kesan

- [Radius Sempadan](./visual/border-radius) - `rounded:small`, `rounded:big`, `rounded:round`
- [Bayang Kotak](./visual/box-shadow) - `shadow:small`, `shadow:medium`, `shadow:big`
- [Sempadan](./visual/border) - `border:grey`, `border-w:[1px]`
- [Kelegapan](./visual/opacity) - `opacity:[0.5]`

## Interaktiviti

- [Prefiks Keadaan](./visual/state-prefixes) - `hover:`, `focus:`, `active:`, `disabled:`

## Contoh Pantas

```html
<div visual="bg:white rounded:big shadow:medium text:dark">
  Kad dengan pelbagai sifat visual
</div>
```
