# Kelegapan

Mengawal kelegapan elemen.

## Nilai

| Nilai | Output CSS |
|-------|------------|
| `opacity:[0]` | `opacity: 0` |
| `opacity:[0.5]` | `opacity: 0.5` |
| `opacity:[1]` | `opacity: 1` |

## Contoh

```html
<div visual="opacity:[0.5]">50% kelegapan</div>
<div visual="opacity:[0] hover:opacity:[1]">Fade in pada hover</div>
```
