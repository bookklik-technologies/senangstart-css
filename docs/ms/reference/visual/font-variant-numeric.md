# Varian Nombor Fon

Mengawal ciri fon numerik.

## Nilai

| Nilai | Output CSS |
|-------|------------|
| `normal-nums` | `font-variant-numeric: normal` |
| `ordinal` | `font-variant-numeric: ordinal` |
| `tabular-nums` | `font-variant-numeric: tabular-nums` |

## Contoh

```html
<span visual="tabular-nums">1,234,567</span>
```
