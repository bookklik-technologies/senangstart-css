# Gaya Senarai

Mengawal gaya senarai.

## Nilai

| Nilai | Output CSS |
|-------|------------|
| `list-none` | `list-style-type: none` |
| `list-disc` | `list-style-type: disc` |
| `list-decimal` | `list-style-type: decimal` |
| `list-inside` | `list-style-position: inside` |
| `list-outside` | `list-style-position: outside` |

## Contoh

```html
<ul visual="list-none">
  <li>Tiada bullet</li>
</ul>
```
