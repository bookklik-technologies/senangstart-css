# Word Break

Mengawal tingkah laku pemisahan perkataan.

## Nilai

| Nilai | Output CSS |
|-------|------------|
| `break-normal` | `word-break: normal` |
| `break-words` | `overflow-wrap: break-word` |
| `break-all` | `word-break: break-all` |

## Contoh

```html
<p visual="break-words">Perkataanpanjangdipecahkandisini</p>
```
