# Font Style

Tetapkan gaya fon

## Sintaks
```
visual="[style-value]"
```

## Nilai

| Nilai | CSS Output | Huraian |
|-------|------------|-------------|
| `italic` | `font-style: italic` | Teks italic |
| `not-italic` | `font-style: normal` | Gaya normal |

## Contoh

```html
<em visual="not-italic">Normal style emphasis</em>
```
