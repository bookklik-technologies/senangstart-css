# Border

Tetapkan warna sempadan

## Sintaks
```
visual="border:[color]"
```

## Contoh

```html
<div visual="border:primary border-w:small">Bordered element</div>
```

## Nilai Arbitrari

Sokong nilai tersuai menggunakan sintaks kurungan segi empat:

```html
<div visual="border:[custom-value]">Custom</div>
```

## Responsif

```html
<!-- Contoh responsif -->
<div visual="mob:... tab:... lap:...">
  Kandungan responsif
</div>
```
