# Line Clamp

Memotong teks kepada bilangan baris tertentu.

## Nilai

| Nilai | Output CSS |
|-------|------------|
| `line-clamp:1` | Potong kepada 1 baris |
| `line-clamp:2` | Potong kepada 2 baris |
| `line-clamp:3` | Potong kepada 3 baris |

## Contoh

```html
<p visual="line-clamp:2">
  Teks panjang yang akan dipotong selepas 2 baris dengan elipsis...
</p>
```
