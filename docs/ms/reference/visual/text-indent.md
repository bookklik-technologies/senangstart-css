# Text Indent

Tetapkan inden teks

## Sintaks
```
visual="indent:[value]"
```

## Nilai

| Nilai | CSS Output | Huraian |
|-------|------------|-------------|
| `0` | `text-indent: 0` | Tiada inden |

## Contoh

```html
<p visual="indent:medium">Indented paragraph</p>
```

## Nilai Arbitrari

Sokong nilai tersuai menggunakan sintaks kurungan segi empat:

```html
<div visual="text:[custom-value]">Custom</div>
```
