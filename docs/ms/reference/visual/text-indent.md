# Inden Teks

Mengawal inden baris pertama.

## Nilai

| Nilai | Output CSS |
|-------|------------|
| `indent:small` | `text-indent: var(--s-small)` |
| `indent:medium` | `text-indent: var(--s-medium)` |
| `indent:[2em]` | `text-indent: 2em` |

## Contoh

```html
<p visual="indent:medium">
  Baris pertama perenggan ini diinden.
</p>
```
