# Smoothing Fon

Mengawal rendering fon dengan atribut `visual`.

## Nilai

| Nilai | Output CSS |
|-------|------------|
| `antialiased` | `-webkit-font-smoothing: antialiased` |
| `subpixel-antialiased` | `-webkit-font-smoothing: auto` |

## Contoh

```html
<body visual="antialiased">Fon lebih lancar di macOS</body>
```
