Tailwind Scale Support (tw- prefix)

This demo shows how to use Tailwind's numeric scale with the tw- prefix.

Spacing - Padding (p:tw-*)

Uses Tailwind spacing: tw-1 (0.25rem), tw-4 (1rem), tw-8 (2rem)...

p:tw-2 (8px)
p:tw-4 (16px)
p:tw-6 (24px)

p-x:tw-8 p-y:tw-2
p-t:tw-10

Mixing Semantic & Tailwind Scales

You can mix semantic values (small, big) with tw- values in the same element!

p:medium + m:tw-4 + shadow:tw-lg
p:tw-6 + m:small + shadow:medium
p:big + rounded:tw-2xl + text-size:tw-lg

Debug Info

Open DevTools (F12) and check:

  • Elements → :root - Look for --tw-* CSS variables
  • Computed Styles - Verify properties resolve to correct rem values

CSS Variable Examples:

  • --tw-4: 1rem → 16px
  • --tw-rounded-lg: 0.5rem → 8px
  • --tw-shadow-lg → Tailwind's lg shadow
  • --tw-text-2xl: 1.5rem → 24px
